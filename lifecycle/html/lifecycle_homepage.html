LifeCycle Catalogue

As part of the LifeCycle project, the EU Child Cohort Network Variable Catalogue has been developed.

The aim of the catalogue is to provide an overview of available data in the EU Child Cohort Network in order to facilitate long-term collaborations between existing pregnancy and child cohorts.
